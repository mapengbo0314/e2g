A

Pluggable Architecture

The Harness just writes structured JSON logs. The Slack integration is a separate background service that reads those logs and handles the Slack API logic (offering both modes via config).

B

Native Webhooks

The Harness itself calls Slack webhooks directly during execution. Simpler to build initially, but adds network latency to the agent's core loop.
