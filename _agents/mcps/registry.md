# MCP Registry

## Planned connectors

- `filesystem_reader`: Read and inspect local project files.
- `media_ocr`: Extract visible text from video frames or screenshots.
- `speech_to_text`: Convert spoken explanation to timestamped transcript.
- `repo_writer`: Apply structured edits to the mapped destination files.

## Notes

These are planning placeholders. Actual MCP wiring can be added as the
workflows become executable inside the chosen harness runtime.
